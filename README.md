# Propositum
---
### Descrição do Projeto

O Propositum é um assistente conversacional inspirado na <em>logoterapia</em> de Viktor Frankl. Por meio de perguntas reflexivas, ele ajuda você a <strong>explorar o sentido da sua vida</strong> e a identificar <strong>pistas vocacionais</strong>.

<strong>Limite importante:</strong> ferramenta reflexiva — <em>não substitui</em> acompanhamento clínico ou terapia profissional.


### Contexto do Projeto

O Propositum é uma plataforma web completa, unindo front-end e back-end, inspirada na logoterapia de Viktor Frankl. Seu propósito é oferecer uma experiência acessível e interativa, em que o usuário possa refletir sobre o sentido da vida e identificar pistas vocacionais por meio de diálogos guiados.  

O sistema combina uma **interface inclusiva e responsiva** com um **back-end capaz de gerar interações dinâmicas e armazenar dados de forma segura**, garantindo tanto a usabilidade quanto a confiabilidade. Dessa forma, o Propositum busca integrar tecnologia e reflexão existencial em um espaço digital acolhedor e funcional.


### Tecnologias Utilizadas

- HTML5  
- CSS3  
- JavaScript (vanilla)  
- Fonte: Merriweather (Google Fonts)

### Arquivos principais

- `index.html` — estrutura da interface e marcação semântica.  
- `main.js` — lógica de interação (controle de fontes, contraste, modal, envio de mensagens).  
- `styles.css` — estilos, variáveis de tema e responsividade básica.

### Como usar (rápido)



1.Abra diretamente o link: https://joaomtm.github.io/propositum-server/ ou abra `index.html` localmente.
2. Ajuste o tamanho da fonte com os botões A-/A+.  
3. Ative/desative o modo de alto contraste conforme necessário.  
4. Abra a ajuda pelo botão `?`.  
5. No campo de entrada, digite a mensagem e envie (Enter ou botão Enviar).
